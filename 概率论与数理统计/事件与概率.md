
# 随机现象

- 在一定条件下并不是总出现相同结果的现象，称之为**随机现象**.
- 随机现象具有两个特点：
	- 出现的结果不止一个；
	- 事先我们并不知道哪个结果会出现。
- 尽管随机现象的结果事先不可预知，但是**很多随机现象的发生会表现出一定的规律性**。随机现象的这种规律性称之为**统计规律性**，其正是概率统计的研究对象。

# 随机试验

- 对随机现象进行的实验和观察，称为**随机试验**，常用大写字母 $E$ 来表示.
- 随机试验具有两个特征：
	- 结果具有随机性
	- 可以重复进行

# 样本空间

- **随机试验的每一个可能结果**，称为**样本点**，通常记为$\omega$
- 随机试验的**所有样本点组成的集合**，称为**样本空间**，通常记为 $Ω$.
- 样本空间来自随机试验.

# 随机事件

- 随机试验的**某些可能的结果组成的集合**，即样本空间中的部分元素所组成的集合，称为**随机事件**，简称为**事件**，通常用大写英文字母$A,B,C$等来表示.
- 随机事件的**本质是集合**，是**样本空间** $\Omega$ 的**子集**
- **基本事件**：**只含有一个样本点**的事件.
- **必然事件**：包含**全部样本点**的事件,即$Ω$.
- **不可能事件**：**不含有任何样本点**的事件，用$\phi$来表示.
- 如果某次随机试验出现的结果 $\omega$ 包含在随机事件 $A$ 中，我们就称事件 $A$ 发生了.以集合论的语言来说，即 $w∈A$ .

# 事件间的关系

- **包含**：若事件A发生必然导致事件B也发生，则称事件A包含于事件B，事件B包含事件A或者事件A是事件B的子事件.记为$A\subset B$ .
- **相等**：若事件A与B使得$A\subset B$和$B\subset A$同时满足，则称事件A与B相等或等价，即是同一事件，记为$A=B$.
- **互不相容**：若事件A与B**不能同时发生**则称事件A与B互不相容。

# 事件间的运算

- 并：由事件A与B**至少有一个发生**构成的事件，称为事件A与B的并，记为$A\cup B$.
- 交：由事件A与B**同时发生**构成的事件，称为事件A与B的交，记为$A∩B$或$AB$.
- 差：由**事件A发生而B不发生**构成的事件，称为事件A与事件B的差，记为$A\setminus B$.
- 对立：事件$\Omega \setminus A$称为事件A的**对立事件**，记为$\overline A$.
- 对称差：事件$A\setminus B$和$B\setminus A$的**并**称为事件A与B的**对称差**，记为$A△B$.
- 特殊标记：
	- 当$AB=\phi$ 时，我们将 $A\cup B$ 写为 $A+B$.
	- 当$A\supset B$时，我们将$A\setminus B$写为$A-B$.
- 特殊性质：
	- $A \cup B = B\cup A$, $AB = BA$.
	- 一般地,$A\setminus B≠ B\setminus A$, 但$A△B=B△A$.
	- 事件A与B互不相容当且仅当$AB=\phi$.
	- 事件A与B**对立，则一定互不相容**；**反之不真**
	- $\overline{\overline A}=A$
	- $\overline A=B$ **当且仅当** $AB=\phi$且 $A\cup B=\Omega$ 成立.
	- $\overline{A}=\Omega-A.$
	- $\overline{A}=B$ 当且仅当 $A+B=\Omega.$
	- $A\triangle B=(A\backslash B)+(B\backslash A),A\cup B=(A\triangle B)+(AB).$
	- $\overline{A\cup B}=\overline{A}\cap\overline{B}$
	- $\overline{A\cap B}=\overline{A}\cup\overline{B}$

# 事件域

- 设 $\Omega$ 表示样本空间，$\mathcal{F}$ 是由 $\Omega$ 的部分子集组成的集合类，若$\mathcal{F}$满足:
	- $\Omega\in\mathcal{F}$
	- $A\in\mathcal{F} \Rightarrow \overline{A}\in\mathcal{F}$
	- 对任意的$n\geqslant1$,$\:A_n\in\mathcal{F}\Rightarrow\bigcup_{n=1}^{\infty}A_n\in\mathcal{F}$
- 则称 $\mathcal{F}$ 为样本空间 $\Omega$ 上的事件域，简称为**事件域**.
- $\mathcal{F}_{0}=\{\Omega,\emptyset\}\text{和}\mathcal{F}_{1}=\{A:A\subset\Omega\}\text{都是事件域}.$
- 设 $\mathcal{F}$ 为样本空间 $\Omega$ 上的事件域，则$\mathcal{F}$ **对事件的有限并,有限交,可列交等运算都封闭**


# 概率

- 设$P(\cdot)$ 是定义在 $\mathcal{F}$ 上的**实值函数**,如果其满足下面三条公理:
	- **非负性公理**:$P(A)\geqslant0$
	- **正则性公理**: $P(\Omega)=1$
	- **可列可加性公理**: 若$A_1,\ldots,A_n,\ldots$ 互不相容，则
$$P\left(\sum_{n=1}^{\infty}A_n\right)=\sum_{n=1}^{\infty}P(A_n)$$
- 则称，$P(\cdot)$为**概率测度或概率**
- 本质上，概率是一个集函数，即自变量是集合(事件)，取值为实数的函数.
- 称三元总体$(\Omega,\mathcal{F},P)$ 为概率空间，$\Omega$ 为样本空间,$\mathcal{F}$ 为事件域, $P$ 是定义在$\mathcal{F}$上的概率测度.

# 概率的性质

- 设$(\Omega,\mathcal{F},P)$ 是给定的概率空间,$A,B,C\in\mathcal{F}$,我们有
- $P(\emptyset)=0$
- 对立事件公式：$P(\bar{A})=1-P(A)$
- 有限可加性：若$AB=\emptyset$，$P(A+B)=P(A)+P(B)$
- 可减性：$\text{若}A\supset B,\text{则}P(A-B)=P(A)-P(B)$
- 单调性：$\text{若}A\supset B,\text{则}P(A)\geqslant P(B)$
- 有界性：$0\leqslant P(A)\leqslant1.$
- 加法公式：$P(A\cup B)=P(A)+P(B)-P(AB)$
- 次可加性：$P(A\cup B)\leqslant P(A)+P(B)$
- $P(A\cup B\cup C)=P(A)+P(B)+P(C)-P(AB).$
- 多个事件的加法公式（庞加莱公式）：
$$\begin{aligned}&\text{一般地},\:
\text{若}n\geqslant1,\:A_{1},\ldots,A_{n}\in\mathcal{F},\:\text{则}\\
P\left(\bigcup_{k=1}^{n}A_{k}\right)&=\sum_{k=1}^{n}P(A_{k})-\sum_{i<j}P(A_{i}A_{j})+\sum_{i<j<k}P(A_{i}A_{j}A_{k})\\&+\cdots+(-1)^{n-1}P(A_{1}A_{2}\ldots A_{n}).\end{aligned}$$

# 古典概型

- 样本空间$Ω$是**有限集**
- **每个基本事件的发生是等可能的.**
$$P(A)=\frac{|A|}{|\Omega|}.$$
- 我们不需要具体列出所有的样本点,只需要知道所含样本点的个数即可，所以计算中经常要用到排列组合工具

# 频率方法

- 频率方法是指**在大量重复试验中用频率的稳定值去替代概率**的方法,在日常生活中被经常使用.基本思想是:
	- 随机试验可大量重复进行
	- $n(A)$ 表示$n$次重复试验中事件$A$发生的次数,称$f_n(A)=\frac{n(A)}n$ 为事件 $A$发生的频率.
	- $f_n(A)$会稳定于某一常数(稳定值).
	- 用频率的稳定值作为事件$A$发生的概率

# 几何方法

- 几何方法是指**借助于几何度量**(长度,面积或体积等)来**计算随机事件的概率**的方法．
- 使用几何方法，需要满足两个条件：
	- 样本空间$Ω$是$n$维空间中的有界区域,$L(Ω)>0$.我们用$L(A)$表示$A$的度量．
	- 每个样本点落在某个子区域的概率与该区域的度量大小成正比,与区域的形状和位置无关。
$$
P(A)=\frac{L(A)}{L(\Omega)}
$$
- **零概率事件未必是不可能事件**
