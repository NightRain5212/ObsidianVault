
## 模板

- 在单调递增序列`a`中查找`>=x`的最小数
```cpp
while(l<r) {
	int mid = (l+r)>>1;
	if(check(mid)) r=mid;
	else l=mid+1;
}
cout<<a[l];
```

- 在单调递增序列`a`中查找`<=x`的最大数
```cpp
while(l<r) {
	int mid = (l+r+1)>>1;
	if(check(mid)) l=mid;
	else r=mid-1;
}
cout<<a[l];
```
