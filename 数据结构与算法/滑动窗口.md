
## 定长滑动窗口

- 若窗口左端点`left=i`，窗口长度`len=k`，则右端点为`right = i+k-1`
- 若窗口右端点`right=i`，窗口长度`len=k`，则左端点为`left = i-k+1`

- **维护答案分三步**：**考虑入窗口，更新维护答案，考虑出窗口**
- 维护一个与窗口有关的量：
	- 右端点开始从左往右遍历，将右端点的元素加入窗口，更新相关值，如果左端点`left<0`，则尚未形成长度为`k`的窗口，无需考虑后续步骤。
	- 更新维护的值
	- 将左端点的元素移出窗口，更新相关值。

- 例子模板：计算数组中长度为k的连续子数组元素和的最大值
```cpp
// 计算数组中长度为k的子数组元素和的最大值
vector<int> s;int sum=0;int ans=0;
for(int right=0;right<s.size();right++) {
	// 计入窗口
	sum+=s[right];
	int left = right - k + 1;
	// 窗口未形成
	if(left < 0) {
		continue;
	}
	// 维护答案
	ans = max(ans,sum);
	// 移出窗口
	sum-=s[left];
}
```

## 不定长滑动窗口

- 滑动窗口相当于在维护一个**队列**。右指针的移动可以视作**入队**，左指针的移动可以视作**出队**。
- 不定长滑动窗口主要分为三类：求最长子数组，求最短子数组，以及求子数组个数。

- 模板：无重复字符的最长子串
```cpp
int lengthOfLongestSubstring(string s) {
	int left = 0,right = 0;
	int ans = 0;
	unordered_map<char,int> count; // 统计窗口出现的字符次数
	for(;right<s.size();right++) {
		// 入窗口
		count[s[right]]++; 
		// 若出现重复字符则缩小窗口，直到不出现重复字符
		while(count[s[right]]>1) {
			count[s[left++]]--; // 移除窗口左端点字符，缩小窗口
		}
		// 维护答案
		ans = max(ans,right-left+1);
	}
	return ans;
}
```
