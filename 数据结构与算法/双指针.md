
## 计算两个已排序数组的最小元素差

- 用两个指针 `p` 和 `q` 分别指向两个数组 `a,b` 的开头。
- 则`d = abs(a[p] - b[q])`，可以推出
$$d = max(a[p],b[q]) - min(a[p],b[q])$$
- 要让`d`减小可以使`max(a[p],b[q])`变小，或让`min(a[p],b[q])`变大。
- 若两个数组`a,b`已经按照非递减排序，则当指针向右移动时，元素变大。
- 所以只需让`a[p],b[q]`的最小值对应的指针`+1`同时维护`d=min(d,abs(a[p],b[q])`即可。
- 时间复杂度：$O(n+m)$
```cpp
vector<int> a(n);
vector<int> b(m);
int p=0;int q=0;
int d = abs(a[p]-b[q]);
while(p<n && q<m) {
	d = min(d,abs(a[p]-b[q]));
	if(a[p]<b[q]) {
		p++;
	}else {
		q++;
	}
}

```
