
# 电容

一个电容器的电容$C$为
$$C=\frac{Q}{U}$$
电容是反映**电容器储存电荷本领大小**的物理量。
**电容器的电容取决于电容器本身的结构**（形状，尺寸，电介质的种类等），与它的带电量无关。
单位为法，符号$F$
$$1F = 1C/V$$
$$1\mu F=1\times 10^{-6}F$$
$$1pF = 1\times 10^{-6} \mu F$$

# 常见电容

## 两平行金属板

$S$为两平行板之间相对的表面积，$d$为距离，带电荷分别为$+Q,-Q$，两板之间场强为
$$E = \frac{\sigma}{\epsilon_0}=\frac{Q}{S\epsilon_0}$$
$$U=Ed =\frac{\sigma d}{\epsilon_0}=\frac{Qd}{S\epsilon_0}$$
$$C=\frac{Q}{U} = \frac{S\epsilon_0}{d}$$
## 圆柱形电容器

圆筒长度为$L$，半径为$R_1,R_2$，内表面外表面分别带电荷$-Q,+Q$，
在两桶之间距离轴线$r$处场强为
$$E = \frac{Q}{2\pi\epsilon_orL}$$
电压为
$$U=\int_{R_1}^{R_2}Edr = \int_{R_1}^{R_2}\frac{Q}{2\pi\epsilon_orL}dr=\frac{Q}{2\pi\epsilon_oL}\int_{R_1}^{R_2}\frac{1}{r}dr$$
$$U = \frac{Q}{2\pi\epsilon_oL}\ln \frac{R_2}{R_1}$$
$$C = \frac{Q}{U} =\frac{2\pi\epsilon_oL}{\ln(R_2/R_1)}$$

## 球形电容器

由两个同心球壳$R_1,R_2$组成，则以此类推可以得到
$$C=\frac{4\pi\epsilon_0R_1R_2}{R_2-R_1}$$

## 计算电容的一般步骤

1. 计算两极板之间场强分布
2. 通过场强分布求得两板之间电势差（电压）
3. 通过电容定义式$C=\frac{Q}{U}$，求得电容

# 电容器的联接

## 并联电容器组

$$I_1+I_2+...+I_n=I$$
$$Q_1+Q_2+...+Q_n=Q$$
$$U_1=U_2=...=U_n=U$$
$$C=\sum C_i$$

## 串联电容器组

- 串联时，等效电容小于任何一个电容器的电容，但**可以提高电容的耐压能力**
$$I_1=I_2=...=I_3$$
$$Q_1=Q_2=...=Q_n=Q$$
$$U_1+U_2+...+U_n=U$$
$$\frac{1}{C} = \sum \frac{1}{C_i}$$
# 电介质对电场的影响

极板间**充满电介质所电容器的电容**为真空电容的$\epsilon_r$倍，其中$\epsilon_r \ge 1$为介质的相对介电常数。
$$U = \frac{U_0}{\epsilon_r}$$
$$C=\epsilon_rC_0$$
$$E = \frac{E_0}{\epsilon_r}$$
# 电介质的极化

## 极性分子

- 电介质分子可分为有极和无极两类
- **极性分子**：分子的正负电荷重心相互分离，**具有电偶极矩**
- 非极性分子：分子的正负电荷重心重合

## 极化

把均匀的电介质放到静电场中时，它的分子会受到电场的作用发生变化，最后达到稳定状态。
- 如果电介质由**非极性分子**组成，这些分子都会沿电场方向产生感生电矩。**外电场越强，感生 电矩越大**
- 如果电介质由**极性分子**组成，这些分子的固有电矩受到外电场的力矩作用而沿着外电场取向，由于热运动，**这种取向只能是部分的**。**外电场越强，取向越整齐**。
极化导致**介质边缘出现电荷分布（只有正电荷或负电荷的电荷层）**，而电介质内部呈现电中性。
这种出现在电介质表面的电荷为**面束缚电荷**。**在外电场作用下，电解质表面出现束缚电荷的现象为电介质的极化。**
**外电场越强，电介质表面的束缚电荷越多**。
当外加电场足够强时，电介质分子中的电子发生电离，变成自由移动的电荷，**这种现象为电介质的击穿**。一种电介质所能承受的不被击穿的最大电场强度为**击穿场强**。


# D矢量及其高斯定律

在有电介质的电场中，由于$E = \frac{E_0}{\epsilon_r}$，$E\epsilon_r = E_0$，
$$\oint \epsilon_0\epsilon_r\vec E \cdot d\vec S = \epsilon_0\oint \vec E_0\cdot d\vec S=\epsilon_0\frac{q}{\epsilon_0}=q$$
定义D矢量为**电位移矢量**
$$\vec D = \epsilon_o\epsilon_r\vec E$$
有D的高斯定律，用于求解有电介质的电场中的场强分布
$$\oint \vec D \cdot d\vec S = q$$
# 电容器的能量

当电容器与电源相连时，进行电容器的充电。反之当电容器充当电源与闭合回路连接时，进行电容器的放电.
设放电过程中，两板带电荷$q$，两板之间电势差为$U$，对一个电荷元$dq$从带负电的极板运到带正电的极板，电场力做的功为
$$dA = -Udq=-\frac{q}{C}dq$$
而放电过程中，两极板电荷由$Q$变化到$0$，
$$W =-A= -\int_{Q}^{0}dA = \int_Q^0 \frac{q}{C}dq = \frac{1}{2C}Q^2 $$
- 所以电容器的能量公式为
$$W =  \frac{1}{2}QU = \frac{1}{2}\frac{Q^2}{C}=\frac{1}{2}CU^2$$
# 电介质中电场的能量

电场能量体密度$w_e$为
$$w_e = \frac{W}{V} = \frac{1}{2}\epsilon E^2 = \frac{1}{2}DE$$
其中$\epsilon=\epsilon_0\epsilon_r$，$D=\epsilon E$
电场总能量
$$W = \int w_edV = \int \frac{1}{2}DEdV$$
