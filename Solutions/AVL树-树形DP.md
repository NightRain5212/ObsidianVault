## 题意

**题目名称**：AVL 树

**背景**：  
AVL 树是一种自平衡二叉搜索树，其平衡条件是任意节点的左右子树高度差不超过 1。题目要求将给定的二叉树通过最少的操作转换为 AVL 树。

---

**AVL 树的定义**：  
1. 空树是 AVL 树。  
2. 非空树是 AVL 树的条件：  
   - 左右子树都是 AVL 树。  
   - 左右子树的高度差不超过 $1$（即 \($|h_{\text{left}} - h_{\text{right}}| \leq 1$)）。  

**操作规则**：  
可以通过以下三种操作调整树的结构：  
1. 删除一个叶子节点。  
2. 为左子节点为空的节点创建一个新的左子节点。  
3. 为右子节点为空的节点创建一个新的右子节点。  

**目标**：  
计算最少需要多少次操作，才能将给定的二叉树转换为 AVL 树。

---

**输入格式**：  
- 多组数据，第一行为数据组数 $T$ $( 1 \leq T \leq 10000 )$。  
- 每组数据：  
  - 第一行为节点数 $n$ $( 1 \leq n \leq 2 \times 10^5 )$。  
  - 接下来 $n$  行，每行两个整数 $( l_{si}, r_{si} )$，表示节点 $i$ 的左右子节点编号（$0$ 表示不存在）。  
- 保证所有数据的总节点数 $( \sum n \leq 2 \times 10^5 )$。  

**输出格式**：  
每组数据输出一行，表示最少操作次数。  

---

**示例**：  
假设输入树为：  
```
1
3
2 3
0 0
0 0
```  
可能需要通过添加或删除节点使其满足 AVL 树的条件，输出最少操作次数。
