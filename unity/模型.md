
# 网格（Mesh）

- 模型是由若干个小面围合组成的形状，且是中空的。
- **3D网格就是一个由顶点、边和面构成的数字化框架，它定义了3D物体的形状和结构。**
- 网格确定了3D模型的形状。
- 网格的主要因素：顶点坐标，边，面。
- 网格的多边形数量是一个网格所包含的**面**的总数。它是衡量模型复杂度和细节程度的关键指标。
- 它的**多边形数量**决定了模型的细节和性能消耗，而它的**拓扑结构**则决定了模型能否良好地动画和变形。它的**多边形数量**决定了模型的细节和性能消耗，而它的**拓扑结构**则决定了模型能否良好地动画和变形。

# 材质（Material）

- **网格**是定义了物体**形状和结构**的“骨架”或“铁丝网”，那么**材质**就是覆盖在这个骨架上的“**皮肤**”、“**油漆**”和“**特性说明**”。它决定了这个物体**看起来是什么样子的**，**定义了物体的表面细节**。
- 材质本身并不是一张张图片，而是一个**包含了多种属性和规则的集合**。
- 材质通过一系列控制光线交互的规则和贴图（金属度，粗糙度，环境光遮蔽等属性），赋予网格颜色、质感、光泽和细节，最终让我们看到逼真或风格化的3D物体。

# 纹理（Texture）

- 纹理就是模型的贴图，用来定义物体的表面颜色。
- **纹理是一张或多张2D图像，它们被映射（包裹）到3D模型的表面，为其提供丰富的视觉细节和物理属性信息。** 纹理是一种内存资源，通常表示为一张或多张二维图像（如PNG, TGA格式）。这些图像包含了用于着色模型表面的数据。
- **材质是一个包含着色器逻辑和参数集合的程序；纹理是存储参数值的数据数组；而网格是应用该程序并进行计算的几何载体。** 三者共同作用，才能生成具有复杂视觉外观的3D图像。
